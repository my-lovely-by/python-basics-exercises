# 🐍 Python 基础教程与练习



## 📖 项目简介

本教程是零基础学习 Python 的理想起点。我们从最简单的挑战开始——比如在终端打印一条消息，然后循序渐进，逐步提升难度。

通过完成全部 **20 个交互式练习**，你将系统地掌握 Python 核心基础知识：

| 学习阶段 | 涵盖内容 | 对应练习编号 |
|---------|---------|-------------|
| **输出与变量** | `print` 函数、变量声明、数据类型 | 00 - 04 |
| **用户输入与字符串** | 用户输入处理、字符串拼接 | 05 - 07 |
| **条件判断** | `if` 语句、条件表达式、实际场景应用 | 08.1 - 08.2 |
| **随机数** | `random` 模块的基本使用 | 09 |
| **函数** | 调用函数、定义函数、参数与返回值 | 10 - 12 |
| **循环** | `while` 循环、`for` 循环、FizzBuzz、综合循环练习 | 13 - 19 |

### 核心特性

- 👆 **交互式学习** — 在浏览器中直接完成代码编写和提交
- ✅ **自动评分** — 每个练习配有测试用例，提交后自动验证结果
- 📹 **视频教程** — 关键知识点配有视频讲解，辅助理解
- 📈 **渐进式难度** — 从 Hello World 到循环综合应用，难度逐步递增

## 📂 练习结构说明

每个练习都是一个独立的小型 Python 应用，包含以下文件：

| 文件 | 说明 |
|-----|------|
| **app.py** | 入口文件，你需要在这里编写代码 |
| **README.md** | 练习的具体要求和说明 |
| **test.py** | 自动测试脚本（无需手动修改） |

### 💡 小贴士

> 练习配有自动评分功能，但评分标准较为严格。建议**不要过于在意测试分数**，把测试结果当作参考即可，专注于理解和掌握知识本身。

## 📚 完整练习目录

以下是全部 20 个练习：

1. **00 - Welcome** — 欢迎页面，开启 Python 学习之旅
2. **01 - Console** — 在控制台输出内容
3. **02 - Declare Variables** — 声明和使用变量
4. **03 - Print Variables In The Console** — 在控制台打印变量
5. **04 - Multiply Two Values** — 学习基本运算
6. **05 - User Inputed Values** — 获取用户输入
7. **06 - String Concatenation** — 字符串拼接
8. **07 - Create a Basic HTML** — 用字符串构建 HTML
9. **08.1 - Your First If** — 你的第一个条件判断
10. **08.2 - How Much The Wedding Costs** — 婚礼预算：条件判断的实际应用
11. **09 - Random Numbers** — 使用随机数
12. **10 - Calling Your First Function** — 调用你的第一个函数
13. **10.1 - Creating Your First Function** — 创建你的第一个函数
14. **11 - Create A New Function** — 创建更多函数
15. **12 - Rand From One to Twelve** — 生成指定范围的随机数
16. **13 - Your First Loop** — 你的第一个循环
17. **14 - Create A For Loop** — 创建 for 循环
18. **15 - Looping With FizzBuzz** — 经典 FizzBuzz 问题
19. **16 - Random Colors Loop** — 循环与随机数的结合
20. **17 - Russian Roulette** — 循环的趣味应用
21. **18 - The Beatles** — Beatles 歌词生成器
22. **19 - Bottles Of Milk** — 循环的综合练习
